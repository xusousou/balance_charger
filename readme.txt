#BETAFPV 2-4S BALANCE CHARGER

##产品说明
● 使用XT30类型的电池端口，充电器可以对2-4S电池充电。
● 支持PD3.0/QC3.0等充电协议。
● 即插即用，智能调节。功率最大60W，充电电流最大3A。


##参数
● 型号：2-4S平衡充电器
● 输入连接器：C型
● 输入电压范围：5-20V（不建议使用5V适配器，充电时间过长且可能会充不满）
● 最大输入功率：60W
● 最大输入电流：3A
● 最大充电电流：3A
● 平衡电流：64mA/cell（需要平衡时，电芯容量越大，电压差越大，平衡的时间会很长）
● 平衡精度：理论上<0.01V，实际 < 0.02。（存在测量误差，主要是市面上的万用表等测量设备精度不一，测量偏差也明显）
● 支持电池连接器：XT30，XH2.54/5pin
● 快充协议：PD3.0/QC3.0
● 电池类型：Lipo 2-4S（暂不支持其他类型电池，请勿使用其他类型电池，否则可能会导致电池起火等危险事故）
● 尺寸：
● 重量：

##充电过程
● 支持 Lipo 2-4S电池充电，将XT30接头和平衡接头连接充电器，充电和平衡将同时自动进行。
● 充电过程中指示灯颜色根据电池电压从低（红）到高（绿）色度逐渐变化，充电完成绿灯常亮。
● 只连接XT30头将会报错，红灯闪烁。
● 只连接平衡头，则只进行电芯平衡。


##指示灯状态
RGB灯指示充电器的状态
指示灯状态                          状态说明
红-->绿-->蓝灯交替闪烁               充电器正在初始化，等待初始化完成
七彩渐变                            待机状态，未连接电池
红灯（或者绿灯）渐变慢闪             充电中
绿灯常亮                            电池充电平衡完成
蓝灯渐变慢闪                        电池电芯正在平衡中
蓝灯常亮                            电芯电压平衡完成
红灯快闪                            充电器异常（如连接错误、电池故障等）


##异常状态
充电器或电池异常表现为红灯闪烁。
以下情况将被视为异常：
● 充电器自检不通过，表现为充电器未接电池，接入适配器红绿蓝三灯亮过之后红灯闪烁。
● 电池电芯异常，未接平衡接口或存在电芯电压低于1V，则视为异常，无法充电。
● 电池电压异常，XT30接口电压与平衡头电压不相匹配。


##按键功能
● 目前按键只有调试用以控制串口发数据
